
# Cozy Cat & Blocks Web Animation

## Description

This is a simple HTML, CSS, and JavaScript animation of a cozy cat playing with colorful building blocks. When you click or tap on the cat, it shows affection by creating floating hearts. The animation also includes a swaying tail, blinking eyes, and a fun bouncing block sequence. The page has a calming pastel color palette with smooth animations for a delightful experience.

## Features

- A cute animated cat with smooth body and head movements
- Interactive block animations (bouncing and rotating)
- Floating heart animations when you interact with the cat
- Responsive and mobile-friendly
- Customizable colors via CSS variables

## Files

1. **HTML**: Contains the structure of the page, including the cat and block elements.
2. **CSS**: Styles the cat, blocks, and animations. Keyframe animations are used for smooth transitions.
3. **JavaScript**: Handles interaction with the cat, triggering heart animations when clicked or tapped.

## How to Use

1. Open the provided HTML file in a web browser.
2. Click or tap on the cat to make it show affection by creating heart-shaped particles.
3. The cat's tail sways, and the cat's face changes expressions when interacting with it.

## Customization

- You can modify the CSS color scheme by changing the CSS variables in the `:root` selector.
- The animation speed and keyframe durations can be adjusted to fit your preferences.

## Requirements

- A modern web browser (Chrome, Firefox, Safari, etc.)
